Security Policy
===============

Reporting a Vulnerability
-------------------------
Please do not file public issues for security vulnerabilities. Instead, email the RADAR-base security team at radar-base@kcl.ac.uk with details and a proof of concept if available. We will acknowledge receipt within 2 business days.

Supported Versions
------------------
This repo hosts documentation only. Security issues should target the affected product repositories; we will route responsibly.

Coordinated Disclosure
----------------------
We follow responsible disclosure practices. We will work with you on a disclosure timeline and credit you if desired.
